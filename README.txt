Make your WooCommerce store more useful with Live Product Estimate and Quote option!

== Description ==

This plugin will provide your users to create products Estimates, ask for Quote, Convert the Estimates to Cart, Print or Email them! These features will help you to supercharge your WooCommerce store and bring more leads!

* How to use
Simply search for wc estimate and quote and install and activate it.

== Installation ==

1. Upload the plugin folder after extracting it to the "/wp-content/plugins/(the folder of the extracted plugin)" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
